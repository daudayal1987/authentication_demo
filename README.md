# authentication_demo
Cookie based authentication in NodeJS with following functionalities
1. Register
2. Login
3. Logout
4. Forgot Password 
5. Reset Password
6. Profile Update

# Environment
NodeJS 8 (Express), MongoDB 4.0

# How to Run
1) Copy repository
2) Configure `config.json`
3) Run `npm install`
4) Run `node start.js`
